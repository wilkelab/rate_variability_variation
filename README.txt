This repository contains all of the scripts, data and figures for the rate_variability_variation paper.

data/
	Contains the data on all of the proteins in the study

figures/
	Figures

manuscript/
	The manuscript

pdb_maps/
	Contains the maps that map the sequence of the virus structures to the corresponding alignment
	
scripts/
	Contains all of the analysis scripts
	
viral_alignments/
	Contains the virus alignments from Sharamoradi et al. (2014)

readme_naming_conventions.txt
	A file that contains information on the virus proteins from Sharamoradi et al. (2014)
